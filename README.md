# プライバシーポリシー

最終更新日: 2026-05-14

「たのめーる発注アシスタント」(以下「本拡張機能」)のプライバシーポリシーを以下のとおり定めます。

## 1. 概要

本拡張機能は、株式会社大塚商会が提供する B2B 通販サービス「たのめーる」(通常版 https://www.tanomail.com/ および法人向けの MA たのめーる https://nma.tanomail.com/) のお申込番号注文画面における入力作業を半自動化するためのブラウザ拡張機能です。

本拡張機能は、ユーザーの認証情報や個人情報を **一切収集・保存・送信しません**。

## 2. 取得・保存するデータ

本拡張機能は、以下のデータをユーザーが利用する端末のローカルストレージ (`chrome.storage.local`) にのみ保存します。これらは **外部サーバーには一切送信されません**。

### 2.1 商品リスト

ユーザーがオプション画面で登録した、よく発注する商品の情報:
- お申込番号 (例: `669-3101`)
- 商品名 (任意の表示用テキスト)
- 通常数量
- 有効/無効フラグ

### 2.2 ジョブ状態 (一時的)

発注処理の実行中、ページ遷移を跨いで処理を継続するため、以下を一時的に保存します。処理完了時またはエラー発生時に自動的に削除されます。
- 現在の処理フェーズ
- 処理対象の商品リスト (上記「商品リスト」の一部)
- 開始時刻

## 3. 取得・保存しないデータ

本拡張機能は、以下を **取得・保存しません**:
- たのめーるのログイン ID およびパスワード (ユーザーは Chrome のパスワードマネージャや手動入力で通常通りログインします)
- 注文履歴、決済情報、配送先情報、その他のたのめーる上の個人情報
- ブラウジング履歴、Cookie、その他のサイトデータ
- 個人を特定できる情報 (氏名、メールアドレス、電話番号、IP アドレスなど)

## 4. データの外部送信

本拡張機能は、収集したデータを **第三者やいかなる外部サーバーにも送信しません**。すべてのデータは利用者の端末内に閉じています。

## 5. パーミッションの使用目的

Chrome 拡張機能として要求するパーミッションの利用目的は以下のとおりです。

| パーミッション | 利用目的 |
|---|---|
| `storage` | 商品リストおよびジョブ状態を端末ローカルに保存するため |
| `activeTab` | popup から現在開いているたのめーるのタブの URL を確認するため (有効性チェック) |
| `scripting` | content script が未注入のタブに対してオンデマンドで注入するため (拡張機能の再読み込み直後でも動作させるため) |
| `declarativeNetRequest` | たのめーる上で表示されるマーケティング用ポップアップ (karte.io) の通信をブロックし、自動操作の妨害を防ぐため |
| `host_permissions: https://www.tanomail.com/*`, `https://nma.tanomail.com/*` | たのめーる (通常版および MA たのめーる) のページ上でのみ動作させるため。他のサイトには一切アクセスしません |

## 6. データの削除

ユーザーは以下の方法で、保存されたデータを削除できます。
- オプション画面で商品を1つずつ削除する
- 拡張機能をアンインストールする (すべての `chrome.storage.local` データが自動的に削除されます)

## 7. 子供のプライバシー

本拡張機能は法人 (大塚商会の「たのめーる」会員企業) の業務用ツールとして提供されており、子供向けに設計・販売されたものではありません。

## 8. プライバシーポリシーの変更

本ポリシーは予告なく改訂されることがあります。重要な変更がある場合は、本拡張機能のストアページまたはアップデートのリリースノートにて告知します。
